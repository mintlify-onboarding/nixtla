## Installation 
You can install [`StatsForecast`](/statsforecast/api-reference/core#statsforecast) with:

```
pip install statsforecast
```

or

```
conda install -c conda-forge statsforecast
```

Vist our [Installation Guide](/statsforecast/getting-started/install) for further instructions.


## Quick Start 

**Minimal Example**

```python 
from statsforecast import StatsForecast
from statsforecast.models import AutoARIMA

sf = StatsForecast(
    models = [AutoARIMA(season_length = 12)],
    freq = 'M'
)

sf.fit(df)
sf.predict(h=12, level=[95])
```

Get Started with this [quick guide](/statsforecast/getting-started/quickstart).

Follow this [end-to-end walkthrough](/statsforecast/getting-started/end-to-end-walkthrough) for best practices.


## Why?

Current Python alternatives for statistical models are slow, inaccurate and don’t scale well. So we created a library that can be used to forecast in production environments or as benchmarks. [`StatsForecast`](/statsforecast/api-reference/core#statsforecast) includes an extensive battery of models that can efficiently fit millions of time series.

## Features
- Fastest and most accurate implementations of [`AutoARIMA`](/statsforecast/api-reference/models#autoarima), [`AutoETS`](/statsforecast/api-reference/models#autoaets), [`AutoCES`](/statsforecast/api-reference/models#autoces), [`MSTL`](/statsforecast/api-reference/models#mstl) and [`Theta`](/statsforecast/api-reference/models#theta) in Python.
- Out-of-the-box compatibility with Spark, Dask, and Ray.
- Probabilistic Forecasting and Confidence Intervals.
- Support for exogenous Variables and static covariates.
- Anomaly Detection.
- Familiar sklearn syntax: `.fit` and `.predict`.



## Highlights

- Inclusion of `exogenous variables` and `prediction intervals` for ARIMA.
- 20x faster than `pmdarima`.
- 1.5x faster than `R`.
- 500x faster than `Prophet`.
- 4x faster than `statsmodels`.
- Compiled to high performance machine code through [`numba`](https://numba.pydata.org/).
- 1,000,000 series in [30 min](https://github.com/Nixtla/statsforecast/tree/main/experiments/ray) with [ray](https://github.com/ray-project/ray).
- Replace FB-Prophet in two lines of code and gain speed and accuracy. Check the experiments [here](https://github.com/Nixtla/statsforecast/tree/main/experiments/arima_prophet_adapter).
- Fit 10 benchmark models on **1,000,000** series in under 5 min.

## Models

### Automatic Forecasting

Automatic forecasting tools search for the best parameters and select the best possible model for a series of time series. These tools are useful for large collections of univariate time series.

| AutoCES | ✅ | ✅ | ✅ | ✅ | 
| AutoTheta | ✅ | ✅ | ✅ | ✅ |
